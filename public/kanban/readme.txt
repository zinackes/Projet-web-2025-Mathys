To get information on how to configure Kanban, explore our documentation: https://docs.dhtmlx.com/kanban/howtostart/.

You can import DHTMLX Kanban into your projects using yarn or npm package manager. To get the trial version of Kanban, run the following commands:

// yarn
yarn config set @dhx:registry https://npm.dhtmlx.com
yarn add @dhx/trial-kanban

Or

// npm
npm config set @dhx:registry https://npm.dhtmlx.com
npm i @dhx/trial-kanban
